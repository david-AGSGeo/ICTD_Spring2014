Octane SDK Documentation

Introduction

The Octane SDK is a .NET library for controlling Impinj Speedway RFID readers.

Requirements

  • Microsoft Windows XP or later
  • Microsoft Visual Studio 2008 or later
  • Microsoft .NET Framework 2.0 or later
  • Impinj Speedway Revolution RFID reader with Octane firmware 4.8.3 or later

The Octane SDK is officially supported on Windows. However, it can run on Linux and Mac OS X under Mono :

http://www.mono-project.com/

The Microsoft .NET Micro Framework and Compact Framework are not currently supported.

Getting Started

  • Create a new Visual Studio project
  • Create a new sub-directory under your project called 'lib'
  • Extract the SDK files (.DLL) to the lib directory
  • Add references to these libraries by selecting Project->Add Reference from the Visual Studio menu
