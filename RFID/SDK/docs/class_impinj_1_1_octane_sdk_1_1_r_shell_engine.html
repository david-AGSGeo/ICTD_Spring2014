Used to open an RShell session with the Impinj reader.

Parameters
addressThe reader hostname or IP address to establish an RShell session with.
usernameThe username used to log on to the Impinj reader with (reader default is "root").
passwordThe password associated with the reader username (reader default is "impinj" for the user "root").
timeoutMsThe timeout period for establishing an RShell connection.
Exceptions
OctaneSdkExceptionThrown when either an RShell connection cannot be established with the Impinj reader, an invalid username or password is provided, or a connection timeout occurs.
