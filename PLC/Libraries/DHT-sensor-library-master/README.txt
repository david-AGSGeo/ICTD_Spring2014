This is an Arduino library for the DHT series of low cost temperature/humidity sensors. 

To download. click the "Download Zip" button on the right, rename the uncompressed folder DHT. Check that the DHT folder contains DHT.cpp and DHT.h. Place the DHT library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.

For a complete installation guide see http://www.freetronics.com/pages/how-to-install-arduino-libraries
